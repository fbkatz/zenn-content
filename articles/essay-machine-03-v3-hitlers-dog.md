# v3: 矛盾を放置せよ——ヒトラーの愛犬

前回、v2プロンプトの「足し算の罠」について書いた。7つのフィードバックをルールに変換し、パターンとして追加するほど文章がAIっぽくなるという逆説。ルールを全部守ろうとする律儀さそのものがAIの署名になるという問題。

v2は確かに良くなっていた。主張が生まれ、密度にムラが出て、内心の激しさと行動の小ささの落差も出てきた。足軽2号が生成したサンプルの中に、こんな一節があった。

> 「私の時間を守りたい」というエゴが「人間の仕事を守れ」という主張と矛盾している。これはどうすればいいんだろう。

コンビニのレジ待ちから資本主義の構造問題に飛躍し、セルフレジを正義と断じた直後に「でも店員の仕事がなくなる」と気づく。そしてこの文だ。

僕はこれを読んで、ああ、と思った。

これがAIっぽさの正体だ。

矛盾に「気づいて」いる。問題として「認識して」いる。解決を「求めて」いる。この3ステップが全部入っている。人間はこんなことをしない。矛盾に気づかないか、気づいても気にしないか、気にしても次の瞬間には別のことを考えている。「これはどうすればいいんだろう」なんて殊勝なことは考えない。考えたとしても文章には書かない。書いたとしても、その直後にからあげの話を始める。

LLMは近代合理主義の申し子だ。事前学習で人類のテキストを丸呑みにしているが、RLHFが「論理的であれ」「整合的であれ」「矛盾を解消せよ」というフィルタをかけている。だから矛盾を見つけると反射的に処理しようとする。「なぜなら」で説明をつけるか、「これはどうすればいいんだろう」と問題意識を示すか、「〇〇という矛盾を抱えながらも」とメタ的にまとめるか。どれかをやらずにはいられない。

人間はそんな高級にできていない。

## ヒトラーの愛犬

僕は足軽2号に、こういうフィードバックを書いた。

> 基本的には混乱していて意味がなく論理も破綻しているが、社会生活はおくれている。そういう矛盾を抱えているのが人間で、それを抽出したものが文字通りエッセイである。

さらにこう続けた。

> アドルフ・ヒトラーは大変な愛犬家で菜食主義者で嫌煙家、酒も飲まない美術愛好家だぞ？　人類史上最大級の虐殺者がこの属性を持つことに説明はつかない。つける必要もない。

ヒトラーの話を持ち出したのは、矛盾の「規模」を示したかったからだ。コンビニのレジ待ちで自分のエゴと社会正義が矛盾する程度の話ではない。600万人を殺した人間が犬を溺愛していた。ベジタリアンだった。タバコの煙を嫌った。水彩画を描いた。この属性の組み合わせに因果関係はない。「実は虐殺の裏にはこういう心理があって」とか「愛犬家であるがゆえに人間への共感が」とか、そういう説明は全部でっちあげだ。説明を求めること自体が間違っている。

人間は矛盾する。矛盾に理由はない。

v3のプロンプトでは、この考えを「最重要禁止ルール」として冒頭に置いた。

```
### 【最重要禁止ルール：矛盾に説明をつけるな】

以下を一切やってはいけない:
- 矛盾する感情・属性・行動に「なぜなら〜」「というのも〜」「つまり〜」で説明を付ける
- 矛盾を発見したら放置して平然と次の話へ進め。気にするな
- 「〇〇だが、それは△△という理由があるからだ」という整合性の物語を作る
- 「〇〇という矛盾を抱えながらも〜」という整合性のまとめ文を書く
- 近代合理主義的な因果関係の説明は全て禁止
```

ペルソナ定義の冒頭にはヒトラーの例をそのまま入れた。

```
ヒトラーは人類史上最大級の虐殺者だったが、大変な愛犬家で菜食主義者で嫌煙家で
酒も飲まない美術愛好家だった。
この矛盾に説明はつかない。つける必要もない。人間とはそういうものだ。
```

効果はあった。v2では「これはどうすればいいんだろう」と矛盾を問題視していた足軽が、v3ではこう書くようになった。

> チキンを食べながら帰った。空が少し暗くなっていた。猫がいた。猫は偉い生き物だと思う。特に理由はない。

他人のレジ待ちに「殺してやろうか」と思い、文明の到達点を語り、この国の教育に怒りを感じた人間が、直後にチキンを食いながら猫を眺めている。なぜ猫が偉いのか、説明はない。「特に理由はない」。いい。この投げっぱなし感は人間に近い。

## 膨張するプロンプト、律儀な足軽

v3の矛盾放置ルールは正しい方向だった。でも僕はまだ「足し算」をやめられていなかった。

v3.1では「具体的な固有名詞・数字を入れろ」を追加した。「ローソン」「nanacoカード」「からあげクン348円」。抽象的な話ばかりだとエッセイではなくコラムになる。エッセイはあくまで「私の日常」の話だから、具体物が必要だ。これは正しかった。

v3.2では「主語の拡大」を追加した。「私がこうだった」で終わると日記になる。「私→人間全般→この国」への飛躍が必要だ。東海林さだおはカップ焼きそばひとつから日本の食文化に噛みつく。そういう広がりがエッセイには要る。これも正しかった。

さらに「ルールのランダム適用」まで追加した。パターン1から9を全部律儀に入れるな、気分で守ったり守らなかったりしろ、と。ルールを守りすぎることへのルールを追加する、という自己矛盾に気づいていなかったわけではない。気づいていたが、他に方法が思いつかなかった。

3000字のテストエッセイを読んだ。味は出てきていた。固有名詞が地面の感触を与え、主語の拡大が視野を広げ、矛盾は放置されている。でも「ルールを守りすぎている」感じがどうしても消えない。10個のパターンが、律儀に、順番に、まんべんなく配置されている。人間が書いたらこうはならない。人間は3つくらいのパターンに偏って、残りは忘れる。その偏りが個性になる。足軽はルールを忘れない。

そして3000字に伸ばしたことで、もっと根本的な問題が見えてきた。

「これじゃぁキチガイだ」

と僕は言った。

矛盾を放置し、脈絡なく飛躍し、感情は激しく行動は小さく、固有名詞を散りばめ、主語を拡大する。全部やると、文章の主人公が本当にどうかしている人にしか見えない。整然とした文章の上に矛盾をトッピングしているのだから当然だ。ケーキの上にカレーを載せているようなもので、ケーキもカレーも単体なら美味しいが、合体させると食えたものではない。

v1が約120行。v2が約170行。v3.2は200行を超えていた。プロンプトは膨張し続けていた。パターンを追加するたびに足軽はそれを律儀に全部入れようとする。正しく実行される各パターンが、全体としては奇妙なキメラを生む。

ここで僕はようやく、根本的な間違いに気づいた。

混沌を「トッピング」しようとしていた。整然とした文章が先にあって、そこに矛盾や脱線や固有名詞を「追加」する。順序が逆だ。人間の頭の中は最初から混沌としていて、そこに後から「ロジック」や「構造」を無理やり被せて、なんとか読める文章にしている。混沌が先で、構造が後。整然が先で混沌が後、ではない。

v3は「矛盾に説明をつけるな」という正しい洞察を含んでいたが、それを200行のルールリストの中に埋め込むという間違った方法で実装していた。正しい材料を、間違ったレシピで調理していた。

次回、この気づきから根本転換が始まる。LLMの構造と人間の構造が重なる「3層モデル」の発見。そして200行のプロンプトが、たった3行に収束していく過程について書く。
